### Running the Hazard Check Simulation (EDA Playground)

1. Replace the file **`instructionMemory.sv`** with **`instructionMemory_Hazard_check.sv`**.

2. Use the custom testbench **`testbench_Hazard_check.sv`** instead of the default testbench.

3. Run the simulation in EDA Playground.

The expected result of the simulation is provided in **`success_log_Hazard_check.txt`**.
If the simulation output matches this file, the implementation is working correctly.
