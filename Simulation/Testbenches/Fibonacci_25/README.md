### Running the Fibonacci-25 Simulation (EDA Playground)

1. Replace the file **`instructionMemory.sv`** with **`instructionMemory_Fibonacci_25.sv`**.

2. Use the custom testbench **`testbench_Fibonacci_25.sv`** instead of the default testbench.

3. Run the simulation in EDA Playground.

The expected result of the simulation is provided in **`success_log_Fibonacci_25.txt`**.
If the simulation output matches this file, the implementation is working correctly.
